# Calculatrice Scientifique

Une application de calculatrice scientifique robuste et intuitive, entièrement développée en **C#**. Ce projet vise à fournir un outil fiable pour effectuer des opérations mathématiques allant de l'arithmétique de base aux fonctions scientifiques plus complexes.

---

## Fonctionnalités (À adapter selon votre code)

* **Opérations standards :** Addition, soustraction, multiplication, division.
* **Fonctions avancées :** Trigonométrie, logarithmes, exponentielles, racines, etc.
* **Précision :** Gestion rigoureuse des priorités d'opérations et des calculs décimaux.
* **Interface :** Interface construite avec Windows Forms (Winforms).

---

## Technologies et Outils

* **Langage :** C#
* **Environnement :** Visual Studio (Utilise le format de solution moderne `.slnx`)
* **Outils de développement :** Intégration de configurations **GitHub Copilot** (via le dossier `.github/agents`) pour assister la création et la révision du code.
